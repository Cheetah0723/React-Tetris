# 用React+Redux+Immutable做 《俄罗斯方块》。
# Use Tetact, Redux, Immutable to coding "Tetris".
--------------------------------------------

### 安装 | Install
```
npm install
```

### 开发 | Development
```
npm start
```
open [http://0.0.0.0:8080/](http://0.0.0.0:8080/)

### 打包 | Build
## Mac:
```
npm run MacBuild
```
## Windows:
```
npm run WindowsBuild
```
